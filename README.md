# SDR_Telescope
Working on a project to create a heat map of the sky using radios.
Uses an  RTL SDR to pull power levels on a frequency into a CSV file and creates graphs .

HOW TO :
1.Make sure you have correct drivers for your SDR
2. Plugin your RTL-SDR
3.Run observation script
4.Make sure a CSV file has been created
5.Run Heatmap script
